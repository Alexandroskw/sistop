\section{Herramientas Utilizadas}
\begin{frame}{Herramientas de Código abierto}
 \begin{itemize}
  
  \item El proyecto LATIN apunta a realizar libros \emph{libres} de forma colaborativa.
  \item Los autores del libro somos usuarios de software libre.
  \item Por ello utilizamos tanto para el desarrollo del material como para la organización grupal distintas herramientas 
  de software libre.
 \end{itemize}

\end{frame}

\begin{frame}[fragile]{Herramienta de Escritura}
 
 \begin{exampleblock}{Emacs + Org mode}	
  \begin{itemize}
  \item Mucho del material ya estaba escrito utilizando emacs y org mode.
  \item Por ello no utilizamos la plataforma de escritura provista por LATIN.
  \item Utilizando org mode la forma de desarrollo no es WYSIWYG. 
  \item Permite separar el proceso de escritura del proceso de edición.
  \item El material se escribe como un archivo de texto.
  \item Luego éste puede ser exportado a HTML, Latex o PDF.
  \end{itemize}

 \begin{exampleblock}{Desarrollo colaborativo y control de versiones}
  \begin{itemize}

   \item Utlizamos una herramienta de control de versiones llamada \alert{git} para mantener los fuentes del libro.
   \item git permite 
      \begin{itemize}
	\item Colaboración concurrente (a distancia).
	\item Revisar cambios.
	\item Mantener distintas versiones del material.
	\item Obtener estadísticas de modificaciones.
	

      \end{itemize}
      
  \end{itemize}

 \end{exampleblock}

\section{Evaluación y Utilización del Libro}
\begin{frame}{Revisión y Evaluación}
\begin{exampleblock}{Revisión}
 \begin{itemize}
  \item Una vez terminada la etapa de escritura comenzamos la etapa de revisión por parte de los autores.
  \item Este proceso lo realizamos de forma cruzada, es decir, una autor corrigió lo que otro escribió.
  \item Al finalzar esta etapa entregamos el libro a los revisores del proyecto LATIN.
  \item El libro también será publicado por la UNAM por lo cual también fue entregado a revisión por parte de la editorial de dicha Universidad.
   
 \end{itemize}
\end{exampleblock}

\begin{itemize}
 \item Por parte del proyecto LATIN el libro fue aprobado para su impresión.
 \item Por parte de la UNAM los revisores también aceptaron el material aunque sugiriendo algunos agregados (ejercitación, resúmenes, etc).
\end{itemize}


 
\end{frame}

\begin{frame}{Utilización en el dictado de clase}
\begin{exampleblock}{}
 \begin{itemize}
  \item El libro en su versión final fue utilizado tanto en la UNR como en la UNAM para el dictado de las materias correspondientes.
  \item Ha habido interés de otras universidades tanto en utilizar el material como en colaborar en su desarrollo (Universidad de Córdoba y de Río Cuarto - Argentina)
 \end{itemize}

\end{exampleblock}

 
\end{frame}

\section{Modificaciones y mejoras Futuras}
\begin{frame}{Modificaciones y mejoras futuras}
 \begin{itemize}
  \item Al entrega al proyecto LATIN el libro estaba en un estado de madurez para su revisión.
  \item De todas formas seguimos trabajando corrigiendo errores ortográficos e incluyendo los cambios pedidos tanto por la UNAM como por los alumnos de los cursos.
  \item Pretendemos en un futuro cercano:
  \begin{itemize}
   \item Agregar resúmenes al final de cada capítulo.
   \item Apéndice con ejecritación junto con sus resultados.
   \item Generar material multimedia.
   \item Incluir nuevos tópicos en el libro (Señales, sistemas distribuidos, etc).
  \end{itemize}

  
 \end{itemize}

\end{frame}


\section{Conclusiones}
\begin{frame}{Conclusiones}
 \begin{itemize}
  \item El resultado dentro del proyecto LATIN ha sido muy satisfactorio y productivo.
  \item Para el desarrollo del material utilizamos diversas herramientas de software libre.
  \item El material está al nivel de publicaciones del área sin el costo asociado a estas y siendo escrito nativamente en español por 
  docentes del área.
  \item El libro ya ha sido utilizado para dictar los curos correspondientes en las universidades con gran aceptación por parte de los alumnos.
